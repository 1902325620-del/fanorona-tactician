# 更新记录

本项目采用语义化版本号。日期使用 `YYYY-MM-DD` 格式。

## [2.0.0] - 2026-07-12

### 正式发布

- 棋局参谋现已支持迂棋、8×8 英式西洋跳棋和莫里斯九子棋，并为三种棋提供完整规则与本地搜索引擎。
- 完成简体中文/英文双语界面、顶部棋类切换、三档搜索强度、候选棋路、撤销和局面编辑。
- Android 版完善系统栏与摄像头开孔安全区、底部快捷确认、顶部先手切换及三种棋终局弹窗。
- 西洋跳棋支持快速取消与换子，并在强制吃子、连续跳吃或无合法走法时显示明确原因。
- Windows x64 便携版与 Android APK 均为完全离线成品，运行时不联网、不调用 AI 接口且不消耗 Token。
- README 与正式发布说明提供中英双语版本。

## [2.0.0-rc.6] - 2026-07-12

### 跳棋选子优化

- 已选中一枚对手棋子时，点击另一枚具备合法走法的棋子会立即切换操控对象。
- 有吃必吃生效时，点击不能执行跳吃的棋子会显示强制吃子提示，不再静默无响应。
- 连续跳吃开始后尝试切换棋子，会提示必须继续使用当前棋子完成连吃。
- 没有强制吃子但所点棋子无合法走法时，会显示对应原因。

## [2.0.0-rc.5] - 2026-07-12

### 交互一致性

- 三种棋在 Web、PWA、Windows 与 Android 上均默认使用“快速”AI 强度。
- 西洋跳棋在所有平台统一支持再次点击已选棋子来取消选择。
- Web 界面布局保持不变：先手仍使用设置面板内的双按钮，终局仍使用原有面板提示而不弹窗。
- Android 继续使用右上角单按钮切换先手，并保留三种棋的终局弹窗。

## [2.0.0-rc.4] - 2026-07-12

### Android 交互优化

- 西洋跳棋首次选中对手棋子后，再次点击同一棋子即可取消；连续跳吃已经开始后不会误清空路线。
- 三种棋在真实终局后统一显示胜负弹窗和终局原因，并提供查看棋盘与重新开局操作。
- 先手切换移至顶部右侧，合并为一个按钮；点击会切换先手并开始新局。
- 三种棋的 Android 默认 AI 强度改为“快速”，手动选择其他强度后重新开局仍会保留选择。
- 上述交互仅由 Android 原生壳启用，Web、PWA 与单文件版保持原有行为。

## [2.0.0-rc.3] - 2026-07-12

### 修复

- Android 版莫里斯九子棋达成终局后立即显示醒目的胜负提示，并明确说明是“棋子不足三枚”还是“无合法走法”。
- 终局提示提供“查看棋盘”和“再来一局”，既不会错过胜利信息，也能保留复盘最终局面的入口。

## [2.0.0-rc.2] - 2026-07-12

### 改进

- 推荐棋路出现后，手机端将“已在游戏中走完”固定在底部安全区上方，无需越过棋盘和分析面板向下滚动即可确认。

### 说明

- 已通过安卓实机确认顶部摄像头、状态栏与底部导航安全区工作正常。

## [2.0.0-rc.1] - 2026-07-12

### 新增

- 在迂棋之外加入《刺客信条 IV》使用的 8×8 英式西洋跳棋，支持强制吃子、连续跳吃、升王、完整路线录入与本地搜索。
- 加入莫里斯九子棋，覆盖摆子、成磨取子、相邻走子、三子飞行及终局判定，并提供本地 Alpha-Beta 搜索。
- 程序界面完整支持简体中文与英文，语言和三种棋类的切换入口统一放在顶部。
- 棋类切换后保留各自棋局状态，不会因临时核对另一种棋而丢失当前局面。

### 修复

- Android 安全区改由原生 `FrameLayout` 容器处理，并向 WebView 传递已消费类型的窗口边距，替代在 WebView 上直接加内边距的旧方案。
- API 24–29 继续使用系统默认栏位布局；API 30+ 同时处理系统栏和摄像头开孔；CSS `safe-area-inset-*` 保留为后备保护。
- 手机端顶部栏改为稳定的多行布局并保持吸顶，刷新、编辑、撤销、语言和棋类切换在窄屏上仍可操作；底部内容继续避让导航栏。

### 说明

- 这是供本地实机测试的候选版本，确认不同安卓机型的安全区和三种棋规则后再发布正式 `2.0.0` Release。

## [1.1.0] - 2026-07-09

### 修复

- Android 15 及更新系统启用全面屏布局后，使用系统栏与屏幕开孔的动态安全边距，避免标题、按钮、棋盘和底部内容被摄像头、状态栏或导航栏遮挡。

### 改进

- 棋子深浅色现在跟随先手方：我方先手时我方为浅色；对手先手时对手为浅色，敌我颜色自动互换。
- PWA 同步支持 CSS 安全区域，并升级离线缓存版本。
- 增加先手配色与 Android 安全区域的自动化测试。

## [1.0.0] - 2026-07-09

- 首个公开版本。
- 提供完整 9×5 Fanorona 规则、本地搜索引擎、Web/PWA、Windows x64 便携版与 Android APK。
